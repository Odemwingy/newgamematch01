# 签历（Qianli）

中国文化日历 + 每日一签 + AI 自我反思解读的轻量应用。

## 技术栈

- **前端**: Expo + React Native + TypeScript
- **后端**: Supabase (Postgres + Auth + Edge Functions)
- **订阅**: RevenueCat
- **监控**: Sentry + PostHog

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# Web 预览
npm run web

# iOS 预览
npm run ios

# Android 预览
npm run android
```

## 环境变量

复制 `.env.example` 为 `.env` 并填入配置。
